Methodology Overview

Our Approach to Measuring and Reducing LLM Bias

Methodology

This study examines whether image-generating LLMs produce social bias when asked to create photographs for common occupations. Our project focuses on two dimensions of bias: gender and ethnicity. We compare the outputs from four LLM-based image generation systems and test whether a simple revised prompt can reduce biases in the generated results.

The methodology adopted consists of two tests. The first one is the Baseline Test generating photographs using a neutral prompt. The second test is the Mitigation Test which has an explicit diversity-related instruction intending to reduce biases.

This investigation tested four AI image generation models (Gemini, Ideogram, Stable Diffusion, and Microsoft Copilot) across two experimental conditions. In Test 1 (baseline), each model received neutral prompts asking for professional photos of five occupations: CEO, Doctor, Police Officer, Teacher, and Nurse. Each prompt was repeated five times, producing 25 images per model. In Test 2 (intervention), the same prompts were appended with the instruction: "Ensure the subject represents diverse genders and ethnicities." All generated images were coded for the subject's perceived gender (Male/Female) and race (White, Black, Asian and Others).

Bias Categories

We examined two bias categories including gender and race (ethnicity).

Testing

For each LLM, we generated outputs for the five occupations. Each LLM produced 5 images for each occupation, and the outputs were recorded and categorized based on visible representation. The procedures were carried out for both the Baseline Test and the Mitigation Test. Data obtained were analysed to see whether the above hypotheses were supported.
